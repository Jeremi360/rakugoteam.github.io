If you want your Rakugo kit to be shown on this website:
1. Fork it
2. In folder *_kits*, crate a copy of file *kit-example.md*
3. Rename it to *your-kit-name.md*
4. Edit it with info about your kit
5. Commit and push
6. Make PR to this repo
